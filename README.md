# Zarafshan Travel Telegram Bot

Телеграм-бот для новой авиакомпании **Zarafshan Travel** с двумя ключевыми функциями:

1. **Автопостинг в канал по заданному интервалу**.
2. **Генерация PDF-билета** с ручным вводом всех нужных данных.

## Возможности

- Команда `/autopost` запускает автопостинг:
  - указываете интервал в минутах,
  - указываете текст сообщения,
  - (опционально) указываете канал, иначе берётся `DEFAULT_CHANNEL_ID`.
- Команда `/stop_autopost` останавливает автопостинг.
- Команда `/ticket` запускает пошаговый мастер генерации PDF-билета.
- PDF сохраняется в папку `tickets/` и отправляется пользователю прямо в чат.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`:

```bash
cp .env.example .env
```

2. Заполните:

- `BOT_TOKEN` — токен бота из @BotFather
- `ADMIN_IDS` — id администраторов через запятую (только они могут пользоваться командами)
- `DEFAULT_CHANNEL_ID` — канал по умолчанию (например `@zarafshan_official` или `-100...`)

## Запуск

```bash
python bot.py
```

## Команды

- `/start` — приветствие
- `/help` — подсказка
- `/autopost` — настроить автопостинг
- `/stop_autopost` — остановить автопостинг
- `/ticket` — создать PDF-билет

## Важно

- Бот должен быть администратором в канале, куда он публикует посты.
- Для отправки в приватные каналы обычно удобнее использовать `chat_id` вида `-100...`.
